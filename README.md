# Introduction 
The goal of this project is to learn and practice software architecture.

### Stack:
- Python
- FastAPI
- SQLAlchemy
- Alembic
- MySQL
- RabbitMQ

### Dessign patterns:
- Domain-Driven Design
  - Aggregate
  - Factory
  - Repository
  - Value Object
- Clean Architecture
- CQRS
- Domain Events
